# 🧠 SentimAI: Text Sentiment Analyzer

**SentimAI** is a full-stack Python application that analyzes the emotional tone of English text, classifying it as Positive, Negative, or Neutral. This project utilizes Natural Language Processing (NLP) techniques and maintains a persistent history of queries.

## 🚀 Features
- **Sentiment Analysis:** Integrated with the `TextBlob` library for polarity scoring.
- **Local Database:** Persistent storage system using `SQLite3`.
- **Interactive Interface:** Modern dashboard built with `Streamlit`.
- **History Management:** View and manage previous analyses directly from the UI.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **NLP Library:** TextBlob
- **Frontend:** Streamlit
- **Database:** SQLite
- **Version Control:** Git & GitHub
